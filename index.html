Building wildest web applications

Savannah is a JavaScript framework that assists in developing web applications using DOM manipulation methods helpers and simple concepts of client-side development.

Helpers

Helper methods enable ready to be used in various situations in development. Some of these methods are: cookie handling, pick up parameters in the URL, email validation, redirection, Ajax and others.
