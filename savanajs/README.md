# SavanaJS
## Build wildest web applications with Savana.js

### Insert:
```html
<script src="lib/js/savana.min.js"></script>
<link href="lib/css/savana.min.css" rel="stylesheet">
```
### Use:
```javascript
$savana(document).done(function(e) {
  Your code here ...   
});
```
